## Install OpenCV

for Mac OSX (10.9.1)

    % brew search opencv
    % brew tap homebrew/science

    % brew info opencv
    opencv: stable 2.4.7.1

    % brew install opencv

## Install ruby-opencv gem

    % gem install bundler
    % bundle install
